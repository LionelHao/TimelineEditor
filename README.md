# TimelineEditor

TimelineEditor是一个基于FFmpeg的视频时间线编辑器，用于Android平台。

## 功能特点

- 视频时间线编辑
- 支持视频剪辑、拼接
- 基于FFmpeg进行音视频处理
- 提供直观的用户界面

## 技术栈

- Android
- FFmpeg
- C++
- Java

## 项目结构

- `app/src/main/java/com/byteflow/learnffmpeg/` - Java源代码
- `app/src/main/cpp/` - C++源代码，包含FFmpeg相关代码
- `app/src/main/res/` - 资源文件

## 构建与运行

1. 克隆项目到本地
2. 打开Android Studio
3. 导入项目
4. 构建并运行

## 依赖

- FFmpeg
- Android NDK
- CMake
